# Jupyter Notebook Analysis
- This notebook contains the analysis to generate sample table to generated in the `bgcflow_config` folder
- To run this notebook, you will need to install the right conda environment. You can find the yaml file in the BGCflow repo: https://github.com/NBChub/bgcflow/blob/main/workflow/envs/bgcflow_notes.yaml
```
conda env create -f bgcflow_notes.yaml # or use mamba
```
- Activate the environment and run jupyter lab:
```
conda activate bgcflow_notes
jupyter lab
```
- it will then give you a link that you can open in your browser, by default its `localhost:8888`
